# POS Keyboard Shortcuts Guide

## Quick Reference

| Shortcut | Action | Context |
|----------|--------|---------|
| `Ctrl + F` | Focus product search input | Anywhere in POS |
| `Ctrl + Enter` | Proceed to checkout | When cart has items |
| `Escape` | Close checkout modal | When checkout modal is open |
| `Arrow Up/Down` | Navigate customer search results | When customer combobox is open |
| `Enter` | Select highlighted customer | When customer combobox is open |
| `Tab` | Navigate through form fields | In checkout modal |
| `+` / `-` buttons | Increase/decrease quantity | Cart item rows (touch-friendly) |

## Detailed Usage

### Product Search Phase
1. Press `Ctrl + F` to focus the product search input
2. Type product name or scan barcode
3. Product is automatically added to cart

### Cart Management
- Use the `+` and `-` stepper buttons for quick quantity adjustments
- Click the price to edit it inline
- Click the trash icon to remove items

### Checkout Phase
1. Press `Ctrl + Enter` to open checkout modal
2. Use smart cash buttons for quick payment entry
3. Press `Tab` to navigate through form fields
4. Press `Enter` to complete the sale

### Customer Selection
1. Click the customer search field
2. Type at least 2 characters to search
3. Use `Arrow Up/Down` to navigate results
4. Press `Enter` to select highlighted customer
5. Press `Escape` to close the dropdown

## Accessibility Features

- All interactive elements are keyboard-navigable
- Focus indicators meet WCAG 2.2 AA standards
- ARIA labels and roles are provided for screen readers
- High contrast mode support
- Reduced motion support for users with vestibular disorders
